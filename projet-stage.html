Plateforme de recherche de stages & alternances

Développement Web • PHP • UX • Projet réel

Contexte

Stage réalisé au sein de B&C Studio, une agence spécialisée en développement web et SEO. J’ai participé à la création d’une plateforme permettant de connecter étudiants et entreprises pour la recherche de stages et d’alternances.

Objectifs

  • Créer une plateforme fonctionnelle et intuitive
  • Permettre aux entreprises de publier des offres
  • Permettre aux étudiants de postuler facilement
  • Optimiser l’expérience utilisateur

Mon rôle : Développeur Web

J’ai participé au développement complet du site : frontend et backend. J’ai travaillé sur plusieurs fonctionnalités clés en PHP, HTML et CSS.

Missions réalisées

  • Création de formulaires de publication d’offres
  • Développement de la page des offres (affichage dynamique)
  • Mise en place du système de candidature
  • Gestion des sessions utilisateurs
  • Développement de la page "Mes candidats"
  • Gestion des CV (upload / suppression)

Processus

Analyse
Conception
Développement
Tests
Optimisation

Fonctionnalités techniques

  • Système de connexion utilisateur (sessions PHP)
  • Formulaire dynamique avec envoi de données
  • Requêtes SQL sécurisées (insertion candidature)
  • Affichage dynamique des offres depuis la base de données
  • Recherche filtrée (entreprise / catégorie)

Défis rencontrés

  • Gestion des images et stockage serveur
  • Sécurisation des données utilisateurs
  • Affichage dynamique des contenus
  • Problèmes d’hébergement (CV bloqués)

Solutions apportées

  • Stockage des images via serveur SSH
  • Vérification des sessions utilisateurs
  • Optimisation des requêtes SQL
  • Gestion des droits serveur

Compétences développées

PHP HTML / CSS SQL Backend UX Gestion de projet

Outils

  • PHP 8
  • HTML5 / CSS3
  • Base de données SQL
  • Serveur SSH

Numérique responsable

Optimisation des ressources serveur (gestion fichiers, images), structuration du code pour limiter les traitements inutiles et amélioration de l’accessibilité des pages.

Conclusion

Ce stage m’a permis de travailler sur un projet concret en entreprise. J’ai développé mes compétences en développement web et en résolution de problèmes. Cette expérience m’a également appris l’importance du travail en équipe et de la rigueur dans un projet professionnel.
