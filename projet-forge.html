Application RPG Fullstack

Symfony • API REST • React • Gestion de données

Contexte

Projet réalisé en groupe visant à développer une application complète de création et gestion de personnages inspirée de Donjons & Dragons. Le projet est composé de deux applications : un backend Symfony (API + fullstack) et un frontend React.

Objectifs

  • Créer une application fullstack complète
  • Développer une API REST fonctionnelle
  • Créer une interface React moderne
  • Gérer des données complexes (relations, stats, groupes)

Mon rôle

Participation au développement backend et frontend : conception des entités, logique métier et consommation de l’API côté React.

Architecture

Symfony
API REST
SQLite
React
CORS

Backend (Symfony)

  • Gestion des entités (User, Character, Race, Class...)
  • Système d’authentification
  • CRUD complet (admin et utilisateur)
  • Upload d’images (avatar)
  • Fixtures pour peupler la base de données

API REST

  • Routes GET pour races, classes, skills
  • Récupération des personnages avec filtres
  • Détail complet des personnages
  • Gestion des groupes (parties)
  • Réponses JSON structurées

Frontend (React)

  • Affichage des personnages sous forme de cartes
  • Filtres (nom, classe, race)
  • Tri (niveau, nom)
  • Navigation dynamique
  • Gestion des états (loading / erreurs)

Logique métier

  • Système de points (Point Buy - 27 points)
  • Validation des caractéristiques (8 à 15)
  • Calcul automatique des points de vie
  • Relations complexes entre entités

Défis rencontrés

  • Communication entre API et React (CORS)
  • Gestion des relations entre entités
  • Validation des données utilisateur
  • Organisation du travail en équipe

Compétences développées

Symfony React API REST SQL Fullstack Travail en équipe

Numérique responsable

Optimisation des appels API, structuration des données pour limiter les requêtes inutiles et utilisation d’une base SQLite légère adaptée au projet.

Conclusion

Ce projet m’a permis de développer une application fullstack complète. J’ai renforcé mes compétences en architecture logicielle, en développement API et en frontend moderne avec React. Cette expérience m’a également appris à travailler efficacement en équipe sur un projet complexe.
